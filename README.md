# 🌐 S3 Static Website Hosting

This project demonstrates how to host a static website using Amazon S3 with versioning enabled. It showcases how to deploy HTML, CSS, and JavaScript files to an S3 bucket and configure it for public access as a website.

## 🚀 Features
- Static website hosting using AWS S3
- Versioning enabled for file recovery
- Public access configuration
- Simple and cost-effective deployment
- High availability and scalability

## 🛠️ Technologies Used
- AWS S3 (Simple Storage Service)
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
- index.html
- error.html
- assets/
  - css/
  - js/
  - images/

## ⚙️ Setup Steps
1. Create an S3 bucket
2. Enable static website hosting
3. Upload website files
4. Enable public access using bucket policy
5. Enable versioning
6. Access the website using the S3 endpoint URL

## 🌍 Use Case
This project is ideal for hosting:
- Personal portfolios
- Landing pages
- Documentation websites
